Memory Walk — Bundler

Builds the entire game (Phaser engine, source files, photos, music) into a single self-contained memory-walk.html file you can email, drop on a USB stick, or commit to GitHub Pages. No server needed to view it — double-click and play.

Photos are decoded (HEIC→JPEG), rotated, and downscaled to 600 px max at build time, so the runtime needs no decoder libraries. Expect ~20–25 MB output and roughly a minute of build time.
