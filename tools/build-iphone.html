Memory Walk — iPhone Bundler

Produces memory-walk-iphone.html — a single self-contained file tuned for iPhone Safari. Open it via AirDrop / iCloud Drive / Files to play offline.

Differences from the desktop bundle: no runtime fetch/Audio.src shim (those fail on some iOS versions); paths rewritten at build time to embedded data URIs directly; visible error banner; portrait-orientation overlay; Leningrad trimmed to the 20-second clip we actually play. Output is ~15–17 MB.
